# Known Issues

## How to Use

Before upgrading or deploying, check the file for your target ansible-core version. Each entry follows this format:

### [Short Description]
**Symptom:** What you see
**Cause:** Why it happens
**Workaround:** Exact steps to fix
**Affected versions:** x.y.z through x.y.w
**Status:** Open / Fixed in x.y.w

## Files
- `ansible-core-2.16.md` — issues specific to the 2.16.x line
- `ansible-core-2.17.md` — issues specific to the 2.17.x line
